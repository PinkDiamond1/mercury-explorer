# mercury-explorer

> Statechain explorer for mercury

## Install api
Install with [npm](https://www.npmjs.com/):

```sh
$ cd api
$ npm install
$ npm run server
```

## Install client
Install with [npm](https://www.npmjs.com/):

```sh
$ cd client
$ npm install
$ npm start
```

### Running tests

Running and reviewing unit tests

```sh
$ npm install && npm test
```
